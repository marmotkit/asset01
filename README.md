# 資產管理系統

## 專案概述
這是一個基於 React 的資產管理系統，提供會員管理、投資組合追蹤、資產配置等功能的網頁應用程式。

## 目前進度
### 已完成
- ✅ 專案基礎架構設置
  - Webpack 配置
  - React 環境建置
  - Material-UI 整合
- ✅ 主要導航框架
  - 響應式側邊導航欄
  - 主要內容區域配置
  - 路由系統建置

### 待開發功能
1. 會員管理模組
   - 會員註冊/登入
   - 會員等級管理
   - 個人檔案管理

   - 公司簡介

3. 投資管理模組

## 技術堆疊
- 前端框架：React
- UI 框架：Material-UI
- 路由管理：React Router
- 建置工具：Webpack
- 開發語言：JavaScript/JSX

## 本地開發
1. 安裝依賴：

bash
npm install

2. 啟動開發服務器：
bash
npm start

3. 訪問開發環境：

http://localhost:3000
